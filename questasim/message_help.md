# Message user-guide

> verror -help

# Example vopt-13408

> verror vopt-13408

```
vopt Message # 13408:
Code coverage of some IPs such as uvm_*, ovm_*, mti_*, etc. used in designs is disabled by Questa.
These are known IPs that do not add value to coverage closure and are excluded from coverage. 
Use "-coververbose" option in vopt to see the list of IPs excluded from coverage for this design.
Please refer the User's Manual section on "Code Coverage and Verification IP" for more information.
```
